# jwt-auth-bootstrap
A bootstrap model for creating web applications with JWT Authentication.

## Backend
Type the following commands:
```
sudo docker-compose up
```
then start the application by running the @SpringBootApplication class, and the application should be online on the selected server.port depending on the application-xxx.properties where 'xxx' is the application profile.

## Frontend
More on the frontend is explained in its README.md file, but in short, type the following commands:
```
npm install
npm start
```
